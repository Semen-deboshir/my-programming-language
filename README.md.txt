# PixelCode Interpreter

Этот интерпретатор выполняет эзотерический язык программирования, который манипулирует пикселями изображения. Код перемещает «курсор» по пикселям и изменяет их RGB-компоненты.

## Как это работает

1. **Входное изображение** — программа обходит его слева направо, сверху вниз (построчно).
2. **Управляющий код** — строка команд, которая определяет действия.
3. **Выходное изображение** — результат применения всех команд.

## Доступные команды

| Команда | Действие |
|---------|----------|
| `>`     | Переместить курсор на 1 пиксель вправо (по `j`) |
| `<`     | Переместить курсор на 1 пиксель влево |
| `.`     | Вывести в консоль текущие `(R, G, B)` |
| `r`     | Увеличить Red на 1 (по модулю 256) |
| `g`     | Увеличить Green на 1 (по модулю 256) |
| `b`     | Увеличить Blue на 1 (по модулю 256) |
| `d`     | Уменьшить Red на 1 (по модулю 256) |
| `e`     | Уменьшить Blue на 1 (по модулю 256) |
| `n`     | Уменьшить Green на 1 (по модулю 256) |
| `R`     | Вывести значение Red |
| `G`     | Вывести значение Green |
| `B`     | Вывести значение Blue |
| `D`     | Прочитать символ из `inp` и записать его код в Red |
| `E`     | Прочитать символ из `inp` и записать его код в Blue |
| `N`     | Прочитать символ из `inp` и записать его код в Green |
| `[`     | Начало цикла |
| `]`     | Конец цикла (переход на `[`, если `(R+G+B) != 255`) |

> **Важно:** Цикл работает в обратную сторону от классического Brainfuck — он продолжается, **пока сумма RGB ≠ 255**.
